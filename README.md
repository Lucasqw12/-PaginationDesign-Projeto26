# 🔢 PaginationDesign

Projeto **PaginationDesign** desenvolvido para criar um componente de paginação moderno e reutilizável utilizando apenas **HTML e CSS**. O objetivo é apresentar uma navegação entre páginas com design limpo e organizado.

## 📌 Sobre o Projeto

O **PaginationDesign** é um componente visual de paginação que simula navegação entre páginas (1, 2, 3, próxima, anterior). O projeto foi criado com foco em praticar estilização, alinhamento de elementos e criação de componentes reutilizáveis.

Este projeto foi desenvolvido utilizando apenas **HTML e CSS**, sem JavaScript.



## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
